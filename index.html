OXSystemsOS

pacman repository

Repositório oficial de pacotes Arch Linux para o OXSystemsOS. Hospedado no GitHub Pages com deploy automatizado.

Início Rápido

Adicione o repositório ao seu pacman.conf e comece a instalar.

terminal 
# Adicionar repositório ao pacman.conf
sudo tee -a /etc/pacman.conf <<EOF
[oxsystemsos]
SigLevel = Optional TrustAll
Server = https://devluanpereira.github.io/oxsystemsos-repo/$arch
EOF

# Sincronizar e instalar pacotes
sudo pacman -Syy

Por que usar?
