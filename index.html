Instalação rápida

Adicione o repositório ao seu sistema, importe sua chave GPG e sincronize os pacotes. Esta configuração é a forma mais consistente para um repositório assinado.

1. Importar a chave do repositório

Troque SEU_KEY_ID_AQUI pelo ID real da sua chave pública GPG.

gpg setup 
sudo pacman-key --recv-keys SEU_KEY_ID_AQUI
sudo pacman-key --lsign-key SEU_KEY_ID_AQUI

2. Adicionar ao pacman.conf

Acrescente a seção abaixo ao final do arquivo /etc/pacman.conf.

pacman.conf 
[oxsystemsos]
SigLevel = Required DatabaseOptional
Server = https://devluanpereira.github.io/oxsystemsos-repo/$arch

3. Sincronizar o banco de pacotes

Atualize os índices para que o Pacman reconheça o novo repositório.

sync 
sudo pacman -Syy

4. Instalar um pacote

Depois da sincronização, já é possível instalar os pacotes publicados.

install 
sudo pacman -S nome-do-pacote

Verificação após a instalação

Use estes passos para confirmar que o repositório foi adicionado corretamente e que a sincronização funcionou.

1

Confirme o repositório

Abra o /etc/pacman.conf e verifique se a seção [oxsystemsos] está presente.

2

Sincronize novamente

Execute sudo pacman -Syy para garantir que a base de pacotes foi recarregada.

3

Teste uma instalação

Instale um pacote publicado no repositório para validar download, assinatura e integração com o Pacman.
